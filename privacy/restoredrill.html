RestoreDrill Backup Verify Privacy Policy

This policy explains how Yohan Labs handles data for RestoreDrill Backup Verify. Contact azij781123@gmail.com with privacy questions or deletion requests.

Shopify permissions

read_themes, read_products, read_online_store_pages, and read_online_store_navigation.

Data processed

Theme file content and metadata, product/page/navigation metadata visible through the requested scopes, coverage information, drill results, and merchant-entered backup vendor reminder details.

Data stored

Theme archives in Cloudflare R2; D1 records for snapshot runs, theme snapshots, file checksums, theme diffs, resource snapshots, coverage reports, coverage map rows, drill reports, reminder settings, alert channel labels, alert events, and export metadata.

Data not stored

Customer data, orders, fulfillments, checkout data, payment data, and live-store write changes.

Retention and uninstall

Retention defaults to 90 days. The retention job deletes expired theme snapshot rows, theme files, theme diffs, drill reports, completed snapshot runs, and associated R2 archive objects. On uninstall or shop redaction, the shop is marked uninstalled. Contact support for deletion requests.

Shopify GDPR webhooks

Customer data request, customer redaction, shop redaction, and app uninstall webhooks are implemented and HMAC-verified. Customer data requests return that no customer data is stored.

Subprocessors and infrastructure

The app runs on Cloudflare infrastructure. Data may be stored in Cloudflare D1 and, for RestoreDrill theme archives only, Cloudflare R2. Alert or email delivery providers are used only where the app feature requires them and the merchant configures that channel.
